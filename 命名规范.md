# 命名规则

## 项目命名
全部采用小写方式，以贯穿线（`-`）分割。
例：my-project

## 目录命名
参照项目命名规则。
有复数结构时，采用复数命名。
例：images, assets, git-hooks

## JS 文件命名
参照项目命名规则
例：event-type.js

## CSS,LESS 文件命名
参照项目命名规则
例：common.less, trip-list.less

## HTML 文件命名
参照项目命名规则
例：index.html, forget-password.html

## Vue 和 React 组件文件命名规范
全部采用驼峰方式，首字母大写。
例：App.vue, TripList.vue, AdvancedProfile.jsx
